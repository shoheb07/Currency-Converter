# Currency-Converter
The Currency Converter Web Application is a simple front-end project built using HTML, CSS, and JavaScript that allows users to convert an amount from one currency to another using real-time exchange rates.

The application provides a clean user interface where users can:

* Enter an amount
* Select source and target currencies
* Instantly view the converted result

It uses an external API to fetch live exchange rates, ensuring accurate and up-to-date conversions.


Key Features

* Real-time currency conversion using API
* User-friendly interface
* Input validation (prevents invalid values)
* Multiple currency options (USD, INR, EUR, GBP, etc.)
* Responsive design (works on mobile and desktop)


Technologies Used

* HTML – Structure of the web page
* CSS – Styling and layout
* JavaScript – Functionality and API handling

How to Use

1. Enter the amount you want to convert
2. Select the currency you are converting from
3. Select the currency you are converting to
4. Click on the Convert button
5. View the result instantly


API Used

https://api.exchangerate-api.com


Installation

1. Download or clone the repository
2. Open the project folder
3. Run index.html in your browser
